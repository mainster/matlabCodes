1. Auf der vorliegenden Diskette "Diskette zum Buch DIGITALE SIGNAL-

   VERARBEITUNG von D. von Gruenigen" finden Sie die vom Autor ent-

   wickelten M-Files und zusaetzliche Funktionen fuer die digitale

   Signalverarbeitung. Die im Buch dokumentierten M-Files wurden

   fuer die MATLAB DOS-Version 3.5 geschrieben und nachtraeglich

   fuer die neue MATLAB Version 4.0 unter Windows angepasst.



   Damit die M-Files auf dieser Diskette ausgefuehrt werden koennen,

   muessen folgende Softwarepakete auf Ihrem Rechner lauffaehig

   installiert sein:



	- MATLAB 3.5 und Signal Processing Toolbox 1.0

   oder

        - MATLAB 4.0 und Signal Processing Toolbox 2.0



   Beide Produkte werden von der bei Punkt 4 angegeben Firma ver-

   trieben und unterstuetzt. (Die MATLAB-Version 4.0 und die

   Signalprocessing Toolbox 2.0 werden erst seit dem Fruehjahr 1993

   vertrieben.)



   Obwohl die Programme auf einem PC geschrieben wurden, koennen die 

   M-Files (ASCII-Files) mit geeigneten Konvertierungsprogrammen auf

   andere Plattformen uebertragen werden (z.B. Macintosh, Sun, VAX, usw). 

   Die MATLAB Versionen 3.5 und 4.0 sind nahezu fuer alle Rechner-

   plattformen erhaeltlich. 



   Die neue Signal Processing Toolbox 2.0 enthaelt einige wesentliche

   Verbesserungen, z.B.: - Die Filterapproximationsfunktionen funk-

                           tionieren jetzt auch fuer strenge 

                           Anforderungen und ergeben kleinere

                           Filterordnungen.

                         - Die Korrelationsfunktionen basieren

                           neuerdings auf einem schnellen Algorithmus

                           und verlangen die Eingangsvektoren in der 

                           korrekten Reihenfolge.

   Die ueberarbeiteten M-Files fuer die MATLAB Version 4.0 tragen

   diesen Verbesserungen Rechnung und funktionieren deshalb nicht

   vollstaendig richtig, wenn die alte Signalprocessing Toolbox 1.0

   verwendet wird.

   



2. Die M-Files auf dieser Diskette werden gleich anderen MATLAB 

   Toolboxen installiert:

  	

	- Erstellen eines Verzeichnisses (z.B MAT35DSV oder MAT40DSV)

          im Verzeichnis, in dem sich das MATLAB Programm befindet 

          (normalerweise c:\matlab).



	- Kopieren aller MAT35DSV- oder MAT40DSV-Files von der 

          Diskette in das entsprechende Verzeichnis.



	- Zusaetzlich muss dem Programm MATLAB der Name des neuen 

          Verzeichnisses mitgeteilt werden. Wie dies zu tun ist, 

          entnehmen Sie bitte Ihrem MATLAB-Handbuch.

         

   Danach koennen die M-Files wie andere MATLAB M-Files aufgerufen

   werden.





3. Bei den vom Autor entwickelten M-Files wurde auf eine Ueberpruefung

   der Eingabedaten weitgehend verzichtet, so dass eine inadequate

   Dateneingabe zum Absturz des jeweiligen Programmes fuehren kann.

   Bei einer unerwuenschten Funktionsweise kann ein Programm

   mit der Tastenkombination CTRL-C unterbrochen und danach neu 

   gestartet werden.

   Da die M-Files-Versionen 3.5 fuer DOS und die M-Files-Versionen 4.0

   fuer Windows entwickelt wurden, koennen fuer andere Betriebssysteme

   entsprechende Anpassungen empfehlenswert sein.
