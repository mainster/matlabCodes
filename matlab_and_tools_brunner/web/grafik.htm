ACDSee
Lies sehr schnell Bilder von beliebigen Laufwerken ein und zeigt diese in verkleinerter Form als Miniaturbild an. Unterstützt werden sämtliche gängigen Grafikformate bis hin zu animierten GIFs.

BildShop Pro 7.6d
BildShop ist die deutschsprachige Version des amerikanischen Programms VuePrint von Hamrick Software. Das Programm bietet die übliche Standardpalette an Funktionen einer Bildbearbeitung, nicht mehr, nicht weniger. Immerhin sind einige außergewöhnliche Features beim Dekodieren von Grafikformaten mit von der Partie. Bilder lassen sich per Tastendruck als Tapete auf den Desktop kleben. Bildschirmschoner können ebenfalls mit Motiven bestückt werden und bieten interessante Überblendeffekte und frei definierbare Hintergrundfarben.

ImageForge Pro 2.9
Zur Bearbeitung von Grafikdateien bietet sich ImageForge PRO an. So lassen sich etwa mit Hilfe dieser Software Fotos retuschieren. Mit den vorhandenen Möglichkeiten können Sie ferner Animationen erzeugen. Das Programm eignet sich auch hervorragend für Grafiken, die zur Verwendung in Web-Seiten vorgesehen sind.

Zum Ordner ImageForge wechseln

IrfanView 3.25
Das kostenlose Kultprogramm Irfan View bietet alle wichtigen Funktionen rund ums Bild, die im PC-Alltag von Bedeutung sind. Das Tool unterstützt zahllose Bild- und Multimedia-Formate und stellt eine übersichtliche Thumbnail-Vorschau von allen Motiven in einem Ordner zusammen. Per Doppelklick auf eine Vorschau wird das Bild in der Originalauflösung angezeigt.
