System Utilities 1.0
System Utilities vereint eine Sammlung nützlicher Programme. Mit ihnen können Sie Ihr System optimieren, mehr Festplattenspeicher schaffen, Laufwerke spiegeln, Sicherheitskopien wichtiger Dateien anlegen und in einem Betrachter Grafikdateien vieler gängiger Formate sichten

Registry Optimizer 4.0
Das Programm erlaubt die einfache Optimierung der Registrierdatenbank von Windows 95/98. Durch Installieren von Dateien wird Speicherplatz beansprucht, der manchmal nach der De-Installation des Programms nicht frei gegeben wird. Registry Optimizer überprüft und optimiert diese Einträge. In der Praxis bringt das eine Performancesteigerung und beschleunigt den Windows-Start.

Cachemanager 4.0
Cachemanager ist ein Programm, das den Disk-Cache (wird auch virtueller Cache oder Dateipuffer genannt) von Windows so optimiert, so dass der PC danach unter Windows mehr freies RAM zu Verfügung haben sollte. Hierzu legt der Cachemanager in den Systemdateien fest, wie groß der Cache sein soll. Dadurch wird dieser nicht gößer als benötigt und Sie gewinnen mehr freies RAM für andere Anwendungen.

DS-Backup 1.2
Sicher ist sicher. Wer wichtige Daten auf der Festplatte speichert, sollte sie regelmäßig auf einen anderen Datenträger sichern. Für den Fall einer destruktiven Virenattacke, eines plötzlichen Festplattencrashs oder eines Systemabsturzes mit in der Folge zerstörten Dateien. Ein Backup ist schnell gemacht vor allem, wenn er vollautomatisch im Hintergrund durchgeführt wird. DS-Backup stellt alle wichtigen Funktionen bereit, die zur Programmierung eines persönlichen Backup-Sets wichtig sind.
