Feurio! 1.50
Feurio!® ist eine integrierte Software zum Erstellen von Audio-CDs (speziell von Samplern) für PCs mit Windows NT® oder Windows 95/98/2000®. Feurio!® vereint ein "Audio-Grabbing" (digitales Auslesen einer Audio CD)-Programm, eine CD-Datenbank (inkl. CDDB-Unterstützung), einen CD- und Wave-Player, einen einfachen Wave-Editor und einen CD-Brenner.

MP3 CD Organizer 2.7
Dieses Programm bringt Organisation in die MP3-Sammlung. Zunächst durchsucht es Festplatten und CDs nach entsprechenden Musikdateien, anschließend werden diese katalogisiert und können vom Anwender beliebig sortiert werden. Direkt aus dem Programm lassen sich Playlists mit den Lieblingsstücken anlegen, die über die Frontend-Unterstützung mit WinAmp wieder- oder als HTML-Dokument ausgegeben werden können.
