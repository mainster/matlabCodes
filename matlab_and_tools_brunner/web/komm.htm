WebZIP 3.71
WebZIP versteht sich als leistungsstarker Online-Staubsauger. Das preisgekrönte Programm saugt ganze Web-Sites aus dem Internet und speichert sie als lokale Kopie auf der Festplatte. Dabei werden alle Links automatisch so abgeändert, dass der Anwender die Seiten später offline problemlos in seinen Browser laden kann - gerne auch in den internen Browser der Shareware. Auf diese Weise lassen sich besonders textlastige Web-Seiten Gebühren sparend offline lesen, zumal die Update-Funktion es ermöglicht, etwa News-Seiten erst zu beziehen, um sie dann täglich per Buttondruck aufzufrischen.

WebZIP Installation starten

AutoEXE 1.0
AutoEXE zeigt Ihnen, wie Sie mit Visual Basic oder Visual C++ selbstextrahierende Archive erzeugen können. Das Programm unterstützt die Archivformate ZIP und CAB. Die Benutzeroberfläche lässt sich für beliebig viele Sprachen anpassen. Eine Reihe von Optionen, wie das Einfügen frei definierbarer Readme-Texte und Lizenzhinweise in das Archiv, runden diese Software ab.
